服务器承载估算：
- 服务器拆分后，单服功能性单一，测试每个API独立处理请求耗时。
## CPU瓶颈（木桶短板）
- 单位时间（1s） / API耗时（0.0000025s） = 40w次并发/秒。
- API执行频率 = 1s / tickrate（32次）= 0.03125s/次
- 40w次 * 0.03125s/次 = 1.5w个房间（单进程）
